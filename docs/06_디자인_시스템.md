## 색상

| 토큰 | HEX | 용도 |
|------|-----|------|
| `primary` | `#0D9488` | 브랜드·주요 버튼 |
| `primaryHover` | `#0F766E` | hover·강조 |
| `primaryMuted` | `#CCFBF1` | 연한 배경 |
| `secondary` | `#334155` | **크롬**(사이드바 등) |
| `neutral50` | `#F8FAFC` | 페이지 배경 |
| `neutral100` | `#F1F5F9` | 테이블 헤더 |
| `neutral200` | `#E2E8F0` | 테두리 |
| `neutral400` | `#94A3B8` | placeholder |
| `neutral600` | `#475569` | 보조 텍스트 |
| `neutral900` | `#0F172A` | 본문 텍스트 |
| `error` | `#DC2626` | 오류 |

`AppColors.surface`, `border`, `textPrimary` 등은 **호환 alias** — 신규 코드는 `neutral*` 사용 권장.

### 색 사용 규칙

| 역할 | 토큰 |
|------|------|
| 크롬(헤더·프레임) | `secondary` |
| 본문·제목 | `neutral900` |
| 보조 텍스트 | `neutral600` |
| 페이지 배경 | `neutral50` |
| 액센트(선택·버튼·포커스) | `primary` |

primary teal은 **넓은 면적 X** — 버튼·선택·포커스 border만.
